<h2>⚠风险提示和免责声明</h2>

此项目仅供学習用，下载后请于24小时内删除源代码和打卡历史记录。

请勿用于实际使用，用于实际使用的一切后果由使用者承担。**打卡默认为您在的状态符合学校要求和防疫规定，若您身体不适/在中高风险、风控区，请手动打卡提交即时数据并及时向属地、辅导员及学校报告！**

请遵守居住地和学校所在地的法律和郑州轻工业大学校规。遵守**郑州轻工业大学违纪处分管理规定之第二章第十九条之第六条规定**
